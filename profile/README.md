## Overview

Project Eyrie is a collection of open-source tools designed to support intelligence, investigative, and OSINT workflows.

The ecosystem includes web applications, lookup tools, mapping workflows, image analysis, email and account research tooling, bookmarklet utilities, and command-line tools built for practical operational use.

These tools are created for lawful use across law enforcement, intelligence, research, security, and private sector investigative environments.

## Privacy and Security

Project Eyrie tools are built with privacy and security in mind.

Where practical, projects are designed to minimise unnecessary server-side processing and keep sensitive handling as close to the user as possible. For web-based tools, this means favouring client-side workflows where possible, reducing unnecessary data retention, and avoiding needless collection or storage of user-supplied material.

Across the ecosystem, the design philosophy emphasises:

- privacy-conscious workflows
- minimised data exposure
- reduced reliance on persistent backend storage where not required
- simple, transparent tooling
- practical security for real operational environments

Where possible, processing is performed locally in the browser or in ways that give users greater control over their data, infrastructure, and workflow.

---

## Design Philosophy

Project Eyrie tools are built to be focused, practical, and operationally useful.

Each project is designed to support real intelligence, investigative, and OSINT workflows with an emphasis on speed, clarity, privacy, security, and direct utility.
